# Express Session with MongoDB

This project demonstrates how to use sessions in Node.js using Express and MongoDB.  
Sessions are stored in MongoDB using connect-mongo.

 Technologies Used
- Node.js
- Express.js
- express-session
- connect-mongo
- MongoDB

 Features
- Create session
- Store session data in MongoDB
- Retrieve session data
- Destroy session
- Cookie expiration (24 hours)
